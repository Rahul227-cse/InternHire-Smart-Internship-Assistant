# InternHire – Smart Internship Assistant

An AI-powered internship tracking and career assistant platform that helps students manage applications, analyze skill gaps, and follow personalized learning paths.

## 🚀 Features

- **Internship Application Tracker**: A clean, Kanban-style board to manage all your applications from 'Applied' to 'Offer'.
- **AI Job Description Analyzer**: Leverages AI to break down job descriptions and highlight what really matters.
- **Skill Gap Detection**: Automatically identifies the skills you're missing for your target roles.
- **Learning Path Generator**: Creates customized educational roadmaps to help you bridge your skill gaps.
- **Internship Readiness Score**: Get a data-driven percentage score on how ready you are for specific internships.
- **Resume Skill Extraction**: Upload your resume in PDF format to automatically extract and analyze your existing skills.
- **Smart Recommendations System**: Personalized internship role suggestions based on your unique skill profile.

## 🛠 Tech Stack

- **Frontend**: Modern HTML5, Semantic CSS3, Vanilla JavaScript (ES6+)
- **Backend**: Python / Flask
- **Database**: SQLite (SQLAlchemy ORM)
- **AI Logic**: NLP-based skill matching and gap analysis

## 💻 How to Run

1. **Clone the repo**:
   ```bash
   git clone https://github.com/Rahul227-cse/InternHire-Smart-Internship-Assistant.git
   cd InternHire-Smart-Internship-Assistant
   ```

2. **Create a virtual environment**:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install requirements**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the application**:
   ```bash
   python app.py
   ```

## 🌐 Deployment

This project is configured for easy deployment on **Render** or **Railway** using Gunicorn.
- **Build Command**: `pip install -r requirements.txt`
- **Start Command**: `gunicorn app:app`
